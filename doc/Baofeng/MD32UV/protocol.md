# Baofeng MD-32UV Protocol


## Device Detection

### Request 

```
 +-----+-----+-----+-----+-----+-----+-----+
 |  'P'  'S'   'E'   'A'   'R'   'C'   'H' |
 +-----+-----+-----+-----+-----+-----+-----+
``` 

### Response

```
 +-----+-----+-----+-----+-----+-----+-----+-----+
 | 06h | 'D'    'P'  '5'   '7'   '0'   'U'   'V' |
 +-----+-----+-----+-----+-----+-----+-----+-----+  
```
